## 🚀 Featured Projects

### 🌾 [Kisan-Mitra](https://github.com/Ayush-Das-9/Kisan-Mitra) *(Python, FastAPI, Groq, Whisper, LLaMA)*
A digital assistant built specifically for farmers. It features an intuitive, numpad-based UI and uses voice interactions. 
- **AI Integration:** Leverages Groq API (LLaMA) for fast reasoning and Hindi TTS for accessible audio feedback.
- **Agentic Workflow:** Processes unstructured agricultural queries and returns tailored actionable insights.

### 📊 [Habit-Tracker](https://github.com/Ayush-Das-9/Habit-Tracker) *(FastAPI, MongoDB, Groq API)*
A smart habit tracking application that goes beyond simple logging by providing AI-powered insights.
- **Agent Integration:** Uses the Groq API to analyze user patterns and provide motivational, personalized feedback.

### 🧠 [RAG Implementation](https://github.com/Ayush-Das-9/Rag_Based_AI) *(FastAPI, Ollama, Whisper, BGE-M3)*
An advanced Retrieval-Augmented Generation (RAG) pipeline designed for video course Question Answering.
- **Architecture:** Utilizes Ollama running Llama 3.2 locally, combined with BGE-M3 for vector embeddings and Whisper for Speech-to-Text.
- **Capabilities:** Processes video content to provide accurate, timestamped, AI-generated answers.
